# Отчёт по лабораторной работе №1

Выполнил: Григорьев Глеб Алексеевич, группа М8О-102БВ-25

## Цель работы

Реализовать бинарное дерево поиска с базовыми операциями, дополнительно реализовать метод нахождения листа с минимальной глубиной (вариант 4)

## Содержание

- `tree.c` — реализация структуры дерева и операций
- `tree.h` — заголовочный файл для импорта и тестирований
- `test.c` — тесты
- `test.h` — заголовочный файл для тестов
- `main.c` — основная программа с текстовым меню
- `Makefile` — сборочный файл

## Как собрать и запустить

### Основная программа

В каталоге с проектом выполнить:

```bash
make all
./main
```

### Тесты
```bash
make test
```
