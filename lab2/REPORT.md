## Основные аспекты программы

### 1. Сортировка Шейкера (Shaker Sort)

**Функция:** `void shaker_sort(int *keys, char **values, int n)`

**Назначение:** Сортировка массива ключей по возрастанию с синхронной перестановкой строковых значений.

| Шаг | Действие | Условие |
|-----|----------|---------|
| 1 | Установка границ | `left = 0`, `right = n - 1` |
| 2 | Прямой проход | Если `keys[i] > keys[i + 1]` — обмен |
| 3 | Сужение правой границы | `right--` |
| 4 | Обратный проход | Если `keys[i - 1] > keys[i]` — обмен |
| 5 | Сужение левой границы | `left++` |

### 2. Двоичный поиск (Binary Search)

**Функция:** `int binary_search(const int *keys, int n, int target)`

**Назначение:** Поиск индекса ключа в отсортированном массиве.

**Входные данные:**
- `keys` — отсортированный массив ключей
- `n` — число элементов
- `target` — искомый ключ

**Выходные данные:** индекс найденного элемента или `-1`, если элемент отсутствует.

| Шаг | Действие | Условие |
|-----|----------|---------|
| 1 | Инициализация диапазона | `left = 0`, `right = n - 1` |
| 2 | Вычисление середины | `mid = left + (right - left) / 2` |
| 3 | Сравнение с `target` | Равенство — вернуть `mid` |
| 4 | Сужение диапазона | Меньше — вправо, больше — влево |
| 5 | Завершение | При пустом диапазоне вернуть `-1` |

### 3. Подготовка входных случаев

**Функции:** `make_sorted_case`, `make_reverse_case`, `make_random_case`

**Назначение:** Формирование трёх вариантов таблицы для запуска сортировки и анализа поведения алгоритма.

| Функция | Назначение |
|---------|------------|
| `make_sorted_case(...)` | Копирует и сортирует таблицу по возрастанию |
| `make_reverse_case(...)` | Копирует, сортирует и разворачивает таблицу |
| `make_random_case(...)` | Копирует и перемешивает таблицу случайным образом |
